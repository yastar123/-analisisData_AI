2

Kirim Pesan

Ketik pesan Anda di kotak input dan klik tombol kirim atau tekan Enter

3

Unggah File

Klik ikon paperclip untuk mengunggah file yang ingin dianalisis

Fitur API Testing

Analisis Dokumentasi API

Masukkan URL dokumentasi OpenAPI/Swagger untuk menganalisis endpoint yang tersedia

Test Endpoint

  • Pilih metode HTTP (GET, POST, PUT, DELETE)
  • Masukkan URL endpoint
  • Tambahkan headers jika diperlukan
  • Masukkan body request jika diperlukan
